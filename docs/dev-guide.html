Developer Guide

Welcome to the webOS Developer Guide. This guide is divided into the following sections:

  • Developing Enyo Applications provides information on developing applications in JavaScript using the new Enyo framework, an integral part of webOS 3.0. If you're interested in creating apps for the TouchPad, you'll want to start here.

  • Developing Mojo Applications documents the Mojo framework, which shipped with webOS 1.x and 2.x. This is intended mainly as a reference for developers with existing Mojo apps.

  • Developing PDK Apps and Plug-ins will be of interest to developers writing apps in C/C++ or a combination of C/C++ and JavaScript, using the webOS Plug-in Development Kit.

  • Developing JavaScript Services is aimed at those creating background processes that leverage the popular Node.js runtime, webOS's add-ons for Node.js, and the HP Foundations API.

  • Developing Synergy Connectors details the workings of webOS Synergy, which offers a variety of tools for connecting users with their contact, calendar, and messaging data.

  • Using the webOS SDK Tools covers the essential tools used by every webOS developer, ranging from shell commands to the software device emulator.

  • Implementing In-App Payment shows how to add in-app payment to your Enyo or PDK application.

Important Notes

  • To install and test applications on a HP webOS device, you need to first enable developer mode. This is not necessary for the Emulator.

  • Once developer mode is enabled, you can then open a device shell and get full access to the file system.
