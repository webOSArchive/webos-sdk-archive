enyo.InputBinding

Extends  enyo.Binding
From enyo26/enyo/source/ext/InputBinding.js

Description

An enyo.Binding designed to have its source or its target be an enyo.Input. If the enyo.Input has a placeholder, it will be used when there is no value. This is a two-way binding.

A path in which the property of the source to bind from may be found.
