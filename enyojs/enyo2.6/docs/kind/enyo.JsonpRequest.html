enyo.JsonpRequest

Extends  enyo.Object > enyo.Async
From enyo26/enyo/source/ajax/Jsonp.js

Description

enyo.JsonpRequest is an Async task specifically designed to wrap JSONP requests to a remote server. Be sure to read about the use cases for JSONP requests, along with the documentation on Consuming Web Services in the Enyo Developer Guide.

Properties

Default: true

When true, a random number is appended as a parameter for GET requests to (attempt to) force a new fetch of the resource instead of reusing a local cache.
