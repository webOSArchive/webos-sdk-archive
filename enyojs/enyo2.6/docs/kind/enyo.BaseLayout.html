enyo.BaseLayout

Extends  enyo.Layout
From enyo26/enyo/source/ui/BaseLayout.js

Description

enyo.BaseLayout provides a basic layout strategy, positioning contained components with the enyo-positioned layoutClass. In addition, it adjusts the layout when reflow() is called, removing or adding the enyo-fit class for components that have set the fit property.
