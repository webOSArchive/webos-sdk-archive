enyo.TouchScrollStrategy

Extends  enyo.ScrollStrategy
From enyo26/enyo/source/touch/TouchScrollStrategy.js

Description

enyo.TouchScrollStrategy is a helper kind for implementing a touch-based scroller. It integrates the scrolling simulation provided by enyo.ScrollMath into an enyo.Scroller.

enyo.TouchScrollStrategy is not typically created in application code. Instead, it is specified as the value of the strategyKind property of an enyo.Scroller or enyo.List, or is used by the framework implicitly.
