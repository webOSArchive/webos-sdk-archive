enyo.LeftRightArranger

Extends  enyo.Layout > enyo.Arranger
From enyo26/lib/layout/panels/source/arrangers/OtherArrangers.js

Description

enyo.LeftRightArranger is an enyo.Arranger that displays the active control and some of the previous and next controls. The active control is centered horizontally in the container, and the previous and next controls are laid out to the left and right, respectively.

Transitions between arrangements are handled by sliding the new control in from the right and sliding the active control out to the left.

For more information, see the documentation on Arrangers in the Enyo Developer Guide.
