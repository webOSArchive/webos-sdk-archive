Extends  > module:canvas/Control~Control
From /home/jonwise/Projects/enyo26/lib/canvas/src/Shape.js

Description

module:canvas/Shape~Shape is the base kind for shapes that can be drawn into the canvas. It doesn't have a default rendering, but an event handler may call its draw() method.

Kinds derived from this one should provide their own implementation of renderSelf(). If more complex operations are needed for filled mode or outline mode, override the fill() method or the outline() method, respectively.
