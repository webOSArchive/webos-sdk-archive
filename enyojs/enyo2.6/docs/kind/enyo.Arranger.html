enyo.Arranger

Extends  enyo.Layout
From enyo26/lib/layout/panels/source/arrangers/Arranger.js

Description

enyo.Arranger is an enyo.Layout that considers one of the controls it lays out as active. The other controls are placed relative to the active control as makes sense for the layout.

enyo.Arranger supports dynamic layouts, meaning it's possible to transition between an arranger's layouts via animation. Typically, arrangers should lay out controls using CSS transforms, since these are optimized for animation. To support this, the controls in an arranger are absolutely positioned, and the Arranger kind has an accelerated property, which marks controls for CSS compositing. The default setting of 'auto' ensures that this will occur if enabled by the platform.

For more information, see the documentation on Arrangers in the Enyo Developer Guide.
