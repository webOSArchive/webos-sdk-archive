enyo.LocalStorageSource

Extends  enyo.Source
From enyo26/enyo/source/data/sources/localStorage.js

Description

A localStorage source. This kind is only an available on platforms that support the localStorage Web standard.

It is important to note that usage of this source requires that models and collections use their respective url properties (enyo.Model.url and enyo.Collection.url). Any collection that needs to be committed must have a unique url value. Any model that will be committed directly, or within an enyo.Collection, must have a unique primaryKey.
