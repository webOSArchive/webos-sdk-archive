private _measureScrollColumns()→ undefined

Moonstone's data grid list delegate uses scroll column metrics to calculate available space for list controls. These metrics are derived via some LESS calculations, so to avoid brittleness we choose to measure them from the DOM rather than mirror the calculations in JavaScript.

Upon request, we do the measurement here (the first time a scroller is rendered) and cache the values in static properties, to avoid re-measuring each time we need the metrics.
