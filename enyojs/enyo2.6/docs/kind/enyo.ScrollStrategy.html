enyo.ScrollStrategy

From enyo26/enyo/source/touch/ScrollStrategy.js

Description

enyo.ScrollStrategy is a helper kind that implements a default scrolling strategy for an enyo.Scroller.

enyo.ScrollStrategy is not typically created in application code. Instead, it is specified as the value of the strategyKind property of an enyo.Scroller or enyo.List, or is used by the framework implicitly.
