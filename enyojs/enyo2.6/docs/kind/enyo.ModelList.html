protected add(models, idx)→ [ 'Array' ].

Adds models to the list, updating an internal table by the model's primaryKey (if possible) and its euid.

Parameters

  • models
    Type: enyo.Model, [enyo.Model]

    The model or models to add to the list.

  • idx
    This parameter is optional
    Type: Number

    If provided and valid, the models will be spliced into the list at this position.

Returns

An immutable array of models that were actually added to the list.

protected has(model)→ Boolean

    Type: enyo.Model, String, Number

    An identifier representing either the model instance, its primaryKey, or its euid.

protected resolve(model)→ undefined, null, enyo.Model

Given an identifier, attempts to return the associated model. The identifier should be a string or number.

Parameters

  • model
    Type: String, Number

    An identifier (either a primaryKey or an euid).

Returns

If the identifier could be resolved, a model instance is returned; otherwise, undefined, or possibly null if the model once belonged to the list.
