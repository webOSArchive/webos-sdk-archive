enyo.ScrollThumb

From enyo26/enyo/source/touch/Thumb.js

Description

enyo.ScrollThumb is a helper kind used by enyo.TouchScrollStrategy and enyo.TranslateScrollStrategy to display a small visual scroll indicator.

enyo.ScrollThumb is not typically created in application code.

Properties
