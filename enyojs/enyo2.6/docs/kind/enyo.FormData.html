enyo.FormData

From enyo26/enyo/source/ajax/formdata.js

Description

An XHR2">www.w3.org/TR/XMLHttpRequest/} FormData implementation. It is used to send multipart/form-data Ajax requests. The internal enyo.Blob kind is the content provider for file-parts.

Note that in Internet Explorer < 10, both enyo.FormData and enyo.Blob are limited to string content and enyo.Blob may only be instantiated using an array or string.

This implementation is inspired by html5-formdata.
