Options : Object

Configurable benchmark options hash. For more details, see the same properties defined on enyo.dev.Benchmark.

Properties

  • name
    Type: String

    The referenceable name of the benchmark (used for reporting).

  • average
    Type: Boolean

    If true (the default), an average of repeated start/stops for the bench will be calculated.

  • logging
    Type: Boolean

    If true (the default), start and stop messages will be written to the console.

  • autoStart
    Type: Boolean

    If true (the default), the benchmark will automatically start on instantiation.
