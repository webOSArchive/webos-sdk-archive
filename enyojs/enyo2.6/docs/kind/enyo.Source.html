enyo.Source

From enyo26/enyo/source/data/Source.js

Description

This is an abstract base class. A source is a communication layer used by data layer kinds to retrieve and persist data and application state via its abstract API methods.

public create(props)→ enyo.Source

Creates an instance of enyo.Source with the given properties. These properties should include a kind property with the name of the kind of source and a name for the instance. This static method is also available on all subkinds of enyo.Source. The instance will automatically be added to the enyo.sources object and may be referenced by its name.

Parameters

  • props
    Type: Object

    The properties to pass to the constructor for the requested kind of source.

Returns

An instance of the requested kind of source.

Overload this method to handle deletion of data. This method should accept an options hash with additional configuration properties, including success and error callbacks to handle the result.
