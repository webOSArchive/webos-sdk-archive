enyo.Layout

From enyo26/enyo/source/kernel/Layout.js

Description

enyo.Layout is the base kind for layout kinds. Layout kinds are used by enyo.UiComponent-based controls to allow for arranging of child controls by setting the layoutKind property.

Derived kinds will usually provide their own layoutClass property to affect the CSS rules used, and may also implement the flow() and reflow() methods. flow() is called during control rendering, while reflow() is called when the associated control is resized.
