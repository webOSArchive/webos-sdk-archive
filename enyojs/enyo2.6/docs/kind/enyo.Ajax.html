enyo.Ajax

Extends  enyo.Object > enyo.Async
From enyo26/enyo/source/ajax/Ajax.js

Description

A kind designed to expose the native XMLHttpRequest API. Available configuration options are exposed by enyo.AjaxProperties.

This kind does not extend enyo.Component and cannot be used in the components block.

For more information, see the documentation on Consuming Web Services in the Enyo Developer Guide.

public go(params)→ this

Executes the request with the given options. The parameter may be a hash of properties or a string. Both represent the query string, with the hash being serialized and the string being used directly.

var query = {q: 'searchTerm'}; // -> "?q=searchTerm"

To provide a POST body, see enyo.AjaxProperties.postBody.

When the request is completed, it will set the xhrResponse property.

Parameters

  • params
    This parameter is optional
    Type: Object, String

    A string or hash to be used as the query string.

Returns

The callee for chaining.
