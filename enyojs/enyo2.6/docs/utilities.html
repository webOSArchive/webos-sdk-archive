public enyo.macroize(text, map, regex)→ String

Populates a String template with values from an Object. Can be used with custom macro patterns.

The default macro pattern is of the form {$name}.

// returns 'My name is Barney.'
enyo.macroize('My name is {$name}.', {name: 'Barney'});

Dot notation is supported like so:

var info = {
    product_0: {
        name: 'Gizmo',
        weight: 3
    }
};

// returns 'Each Gizmo weighs 3 pounds.'
enyo.macroize('Each {$product_0.name} weighs {$product_0.weight} pounds.', info);

Parameters

  • text
    Type: String

    The template text in which to insert values via macro.

  • map
    Type: Object

    The Object containing the data to be inserted into the text.

  • regex
    This parameter is optional
    Type: RegExp

    The optional pattern to use to match the entries in the text.

Returns

The modified text value.

public enyo.quickMacroize(text, map, regex)→ String

A non-recursing version of enyo.macroize(). This means it will not expand the same macro more than once in the String, but it is much more efficient than the recursing version.

Parameters

  • text
    Type: String

    The template text in which tokens will be replaced via macro.

  • map
    Type: Object

    The Object containing the data to be inserted into the text.

  • regex
    This parameter is optional
    Type: RegExp

    The optional pattern to use to match the entries in the text.

Returns

The modified text value.

public enyo.quickReplace(text, map, regex)→ String

Uses a regular expression pattern to replace tokens in a String. This is just like enyo.macroize(), except it does not support nested structures (i.e., dot notation).

Parameters

  • text
    Type: String

    The template text in which tokens will be replaced via macro.

  • map

    The Object containing the data to be inserted into the text.
