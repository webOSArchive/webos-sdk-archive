Description

An internally-used mixin that is added to enyo.Object and its subkinds. It includes public and protected API methods for working with bindings.

public binding(props)→ this

Imperatively creates a binding. Merges a variable number of hashes and instantiates a binding that will have its owner property set to the callee (the current enyo.Object). Bindings created in this way will be destroyed when their owner is destroyed.

Parameters

  • props
    Type: Object

    A variable number of hashes that will be merged into the properties applied to the enyo.Binding instance.

Returns

The callee for chaining.

public removeBinding(binding)→ this

Removes a single enyo.Binding from the callee. (This does not destroy the binding.) Also removes the owner reference if it is the callee.

It should be noted that when a binding is destroyed, it is automatically removed from its owner.

Parameters

  • binding
    Type: enyo.Binding

    The enyo.Binding instance to remove.

Returns

The callee for chaining.
