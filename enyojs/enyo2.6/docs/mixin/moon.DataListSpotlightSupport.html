moon.DataListSpotlightSupport

From moonstone/source/DataList.js

Description

moon.DataListSpotlightSupport is a mixin that provides Spotlight handling code for use by moon.DataList and moon.DataGridList. Since both of these kinds inherit from their respective Enyo counterparts, this mixin provides the common add-on code needed for proper spotlight handling.
