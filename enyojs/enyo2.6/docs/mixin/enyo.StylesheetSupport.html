MIXIN

enyo.StylesheetSupport

From enyo26/enyo/source/ui/StylesheetSupport.js

Description

The enyo.StylesheetSupport mixin is used to add a "side-car" inline stylesheet to a control, specifically for procedurally-generated CSS that can't live in the more appropriate location (i.e., in a CSS/LESS file).
