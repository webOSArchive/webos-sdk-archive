MIXIN

enyo.MultipleDispatchSupport

From enyo26/enyo/source/kernel/mixins/MultipleDispatchSupport.js

Description

A collection of methods to allow a single enyo.Component to dispatch a single event to multiple targets. The events are synchronously propagated in the order in which the targets are encountered. Note that this mixin is already applied to a base kind, enyo.MultipleDispatchComponent.
