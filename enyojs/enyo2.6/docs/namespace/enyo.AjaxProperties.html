public postBody : String, Buffer, Object
Default: ''

The content for the request body for POST/PUT methods. When postBody is a buffer or a string, it is passed verbatim in the request body. When it is a hash, the way it is encoded depends on the contentType:

  • 'application/json' => JSON.stringify()
  • 'application/x-www-urlencoded' => url-encoded parameters
  • 'multipart/form-data' => passed as fields in enyo.FormData (XHR2 emulation)
