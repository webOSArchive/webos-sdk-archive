public parse(json, reviver)→ Object, null

Wrapper for JSON.parse(). Parses a valid JSON string and returns an object, or null if the parameters are invalid.

Parameters

  • json
    Type: String

    The JSON string to parse into an object.

  • reviver
    This parameter is optional
    Type: function

    The optional function to use to parse individual keys of the return object.

Returns

If parameters are valid, an object is returned; otherwise, null.

public stringify(value, replacer, space)→ String

Wrapper for JSON.stringify(). Creates a JSON string from an object.

Parameters

  • value
    Type: Object

    The object to convert to a JSON string.

  • replacer
    This parameter is optional
    Type: function, [String]

    An optional parameter indicating either an array of keys to include in the final output or a function that will have the opportunity to dynamically return values to include for keys.

  • space
    This parameter is optional
    Type: Number, String

    Determines the spacing (if any) for pretty-printed output of the JSON string. A number indicates the number of spaces to use in the output, while a string will be used verbatim.

Returns

The JSON string for the given object.
