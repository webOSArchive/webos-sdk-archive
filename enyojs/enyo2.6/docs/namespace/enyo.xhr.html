private request(params)→ XMLHttpRequest

Internally-used method to execute XHR requests.

Note that we explicitly add a 'cache-control: no-cache' header for iOS 6 for any non-GET requests to work around a system bug causing non-cachable requests to be cached. To disable this, use the header property to specify an object where cache-control is set to null.
